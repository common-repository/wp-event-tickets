== Installation ==

	= Uploading in WordPress Dashboard =

	1. Go to the 'Add New' in the plugins dashboard
	2. Navigate to the 'Upload' area
	3. Select wp-event-tickets.zip from your computer or Upload `wp-event-tickets.zip` to the `/wp-content/plugins/` directory directly.
	4. Click 'Install Now'
	5. Activate the plugin through the 'Plugins' menu in WordPress.
	6. You will also need to activate WooCommerce plugin for this to work.
